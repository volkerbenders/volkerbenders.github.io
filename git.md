# configure user data
```bash
$> git config --global user.name "Ihr Name"
$> git config --global user.email ihre@emailadresse.de

Nachdem Sie das getan hast, können Sie Ihre Identität für diesen Commit ändern mit:

$> git commit --amend --reset-author
```
